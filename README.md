# Flutter Camera Geolocation 

This is a flutter project to 
- capture image
- get location
- put location info on top of image
- save into gallery

Using libraries
- image_picker
- image
- geolocator
- geocoding
- permission_handler
- get
- intl
- image_gallery_saver
- path_provider
- screenshot
